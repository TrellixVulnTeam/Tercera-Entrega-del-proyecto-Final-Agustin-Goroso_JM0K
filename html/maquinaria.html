Nuestra maquinaria

Ploteadora

Un plotter de impresión (plóter, ploteadora o trazador gráfico) es una impresora de gran formato que se utiliza para imprimir gráficos vectoriales y/o cortar con alta precisión.

Grabadora Laser

El grabado a láser es un proceso que vaporiza los materiales en forma de rayos para grabar marcas permanentes y profundas. El rayo láser actúa como un cincel, grabando marcas al eliminar capas de la superficie del material.

Estampadora

Una maquina estampadora plana, por ejemplo, principalmente es usada para para decorar ropa, carteles y pancartas, así como elementos como fundas para teléfonos móviles, tazas de café, fundas para portátiles y otros elementos con superficie apta para la sublimación.
